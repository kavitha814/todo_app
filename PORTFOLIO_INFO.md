# Portfolio: Todo App - Tasks & Routine Manager

**Description:**
A smart, dual-mode task management application built with Flutter, featuring seamless daily routine tracking, task organization, and real-time cloud synchronization via Firebase.

## How to Install the APK

1.  **Download**: Click the provided link to download `app-release.apk` to your Android device.
2.  **Open**: Locate the file in your "Downloads" folder and tap to open.
3.  **Security Prompt**: If you see a security warning (standard for apps outside Google Play), tap **Settings** and enable **"Allow from this source"**.
4.  **Install**: Tap **Install** and wait for the process to complete.
5.  **Launch**: Open the app and sign in to sync your tasks!
